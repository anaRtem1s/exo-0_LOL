# 🛡️ Liste de Champions - JavaScript

Petit projet JavaScript permettant de gérer une liste de champions favoris (inspiré de League of Legends 🧙‍♂️⚔️).

## 🎯 Objectif

Ce projet a pour but de pratiquer :

- La manipulation du DOM
- Les événements (`click`)
- Les tableaux (`array`)
- La création dynamique d’éléments HTML
- La séparation de la logique (fonctions)

---

## 🧠 Fonctionnalités

✅ Affichage d’une liste de champions existants  
✅ Ajout d’un nouveau champion via un champ input  
✅ Mise à jour dynamique de la liste  
✅ Reset automatique du champ après ajout  
✅ Gestion des erreurs (champ vide)

---

## 🖥️ Aperçu

- Une liste `<ul>` contenant les champions
- Un champ input pour ajouter un champion
- Un bouton pour valider l’ajout

---

## ⚙️ Technologies utilisées

- HTML5
- CSS3 (optionnel pour le style)
- JavaScript (vanilla)
